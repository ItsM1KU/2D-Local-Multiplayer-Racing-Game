

Instructions:

1. Enter 1 or 2 depends if you want to host or join a game.

2. If you are host, you need to provide your Ip address to guest.
   
   Bonus: If you are on same machine 127.0.0.1.
          Different machines but same wifi, then type ipconfig in cmd
  
3. Guest will enter the host ip address and connect to the game.
   
4. Game window should open if the connection is established.

Note: This is a game project made for a university test. This project implements both UDP and TCP for the network architecture and completely made on SFML.
